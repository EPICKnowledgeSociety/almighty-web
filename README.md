# Almighty Web

The next web after web 2.0. Main concepts and requirements.

The main concept:

* Legal validity of digits
* Crypto democracy

Ideas:

* Everythink computable
* Distributable computations
* Device free
* Network Identity
* Addressing (phone numbers or email analogue)
* Personal clusters
* Smart home network
* Cross site interactions


## Technologies

* WASM
* WebRTC
* CRDT
* Blockchain
